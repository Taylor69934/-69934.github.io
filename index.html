「SPARK+」AI 辅助新消费能量饮广告视觉全案

SPARK YOUR DAY. 点燃今日状态。

设计概念 / Concept

SPARK+ 是一款面向 Z 世代的能量气泡饮。设计语言强调“瞬间爆发”的能量感与极度冰爽的感官体验。通过高饱和度的色彩与动感的流体溅射，传达品牌的核心精神。

商业应用 / Application

涵盖了包装系统、主视觉 KV、天猫详情页、直播间背景及社媒矩阵图（Social Media）。并特别输出了 10s 短视频分镜，完成了从静态到动态的全面覆盖。

AIGC 流程 / AI Workflow

展示了高效的 AIGC 商业管线：AI 营销方向探索 → 多场景 AI 生成 → 包装设计与 3D 建模 → 产品与场景合成 → 详情页与社媒排版落地，产出效率提升 300% 以上。

《Future Sense Festival：AIGC辅助青年艺术节主视觉设计》

一个关于 AI、音乐、城市夜游与数字自然的沉浸式青年艺术节视觉系统设计

活动设定 / Event Profile

时间：2025.08.15 – 08.17
地点：上海（西岸艺术中心、沿江夜游区域、工业遗址空间）
受众：18–30岁热爱音乐、数字艺术及社交传播的年轻用户。

活动形式 / Experience

包含音乐演出、AI影像装置、交互艺术展、夜游路线、快闪市集及沉浸式光影空间，打造全方位的城市青年文化体验。

AIGC 工作流 / Workflow

运用 ComfyUI 搭建专属工作流，结合 SDXL 模型与精准的 ControlNet 控制，实现数字自然（Cyber Nature）与液态发光材质的高质感稳定输出。

「雾川 WUCHUAN」城市气味香氛品牌设计

探索东方美学与人工智能生产力的无缝融合

设计概念 / Concept

以“一座城的嗅觉记忆”为切入点，将上海（现代雨夜、老洋房玻璃幕墙）与杭州（西湖水汽、龙井茶山）的地域文化碎片转化为具体的感官视觉符号。

AIGC 工作流 / Workflow

利用 Midjourney 生成极具质感的光影和宣纸/烫金纹理意象；通过 Stable Diffusion 约束瓶身结构与视角稳定性；最后配合 Photoshop 进行高精度的商业排版与色彩管理。

核心价值 / Value

展现了“懂实体产品逻辑（工业设计底子）+ 掌握 AI 顶级生产力”的复合能力，极大地缩短了虚拟消费品牌从概念创意到高精度视觉资产的研发周期。

《凤小啾 FENGXIAOJIU》3D潮玩IP设计与孵化全案

探索数字角色资产向实体潮玩商业转化的全链路盲盒企划

角色设定 / IP Profile

凤小啾，一只拥有幻化能力的数字潮流精灵。以高辨识度的暗金与奇幻紫为主色调，能够结合不同的潮流时空穿梭变换百变造型，传递陪伴与治愈的潮玩理念。

材质与表现 / 3D LookDev

精准把控实体潮玩质感表现，模拟真实的 PVC 与 ABS 材质光影，配合精细的表面高光与哑光对比，使得数字模型具备极高的实物开模参考价值。

商业延展 / Commercial Value

输出了标准工业三视图、包装设计概念，并完成了多维度的主题盲盒系列化延展（如：宇航员、魔法师、新春款等），具备直接对接潮玩供应链的成熟度。

「AURA 光冕」独立首饰品牌视觉与产品设计全案

WE CROWN OURSELVES. 我们为自己加冕。

品牌概念 / Brand Identity

以“光冕”为核心意象，提取光环（Halo）、锋芒（Edge）、盾牌（Shield）等几何符号，塑造女性独立、自我加冕的先锋力量与美学态度。

数字工艺 / Digital Craft

展示扎实的工业结构与产品逻辑。从严谨的底层线框构建（Model View）到正侧俯三视图推敲，并进行高保真渲染，完美还原 925 银的冷硬光泽与宝石的璀璨折射。

AIGC 赋能 / AI Integration

打破传统珠宝宣发的局限，运用 AIGC 工作流高效生成具备极强奢侈品感的高级时尚模特佩戴图（Lookbook）与高端商业空间氛围，极大拓展了数字资产的商业边界。

「NOVA SCALP」智能头皮护理仪产品设计全案

TECHNOLOGY × CARE × WELLNESS. 科技呵护头皮，焕发健康活力。

设计概念 / Design Concept

结合人体工学与现代极简美学，打造流线型握持曲线与优雅的底座收纳，打破传统理疗仪的医疗感，赋予其家居艺术品的呼吸感。

结构解析 / Structure & Tech

深入工业落地逻辑。输出严谨的产品爆炸图与细节解析，涵盖精密梳齿排布、磁吸充电系统及核心驱动模组，展现扎实的硬核设计底子。

材质表现 / CMF LookDev

细腻刻画高级 CMF（色彩、材料、表面处理）。珍珠白机身搭配香槟金边饰，精准还原亲肤磨砂质感与金属光泽的高保真 3D 渲染表现。
